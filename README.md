# 💸 SplitMitra - Expense Split Application

## 📌 Project Description

SplitMitra is a Django-based web application to manage and split expenses among friends.

---

## 🚀 How to Run the Project

### 1. Clone the repository

git clone https://github.com/Pradip9186/SplitMitra.git
cd SplitMitra

### 2. Create virtual environment

python -m venv .venv

### 3. Activate environment

.venv\Scripts\activate

### 4. Install dependencies

pip install -r requirements.txt

### 5. Apply migrations

python manage.py makemigrations
python manage.py migrate

### 6. Run server

python manage.py runserver

---

## 🌐 Open in Browser

http://127.0.0.1:8000/

* Expense splitting
